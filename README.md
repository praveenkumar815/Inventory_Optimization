# Supply Chain Profitability Analysis

## Overview
This project analyzes retail sales data to identify profit bleed and optimize logistics.
It uses a Python ETL pipeline to load data into MySQL, SQL for analysis, and Power BI for visualization.

## Tech Stack
- **Python:** Pandas, SQLAlchemy (ETL Pipeline)
- **SQL:** MySQL (Data Warehousing & Logic)
- **Power BI:** Dashboarding

## Key Insights
- Identified sub-categories causing a 15% margin leak.
- Optimized shipping mode selection to improve logistics efficiency.
